Python packages
Python packages require Python 3.8 or newer and can be installed with pip. 
$ pip install <package>
where <package> is one of:
  • hat-util (docs)
  • hat-peg (docs)
  • hat-sbs (docs)
  • hat-chatter (docs)
  • hat-juggler (docs)
  • hat-duktape (docs)
  • hat-sqlite3
  • hat-drivers (docs)
  • hat-orchestrator (docs)
  • hat-monitor (docs)
  • hat-event (docs)
  • hat-gateway (docs)
  • hat-gui (docs)
  • hat-translator (docs)
JavaScript packages
JavaScript packages can be installed with npm. 
$ npm install <package_url>
Available packages:
  • @hat-core/util
  • @hat-core/renderer
  • @hat-core/future
  • @hat-core/juggler
