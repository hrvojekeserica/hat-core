class hat.duktape.Lib(path=None)¶

Bases: object

Ducktape library

Low-level interface to ducktape directly mapping ducktape functions and constants to python. Only parts used by Interpreter are exposed. For detailed documentation see https://duktape.org/api.html.

Parameters

path (Optional[path]) – dynamic library file path

JavaScript interpreter
